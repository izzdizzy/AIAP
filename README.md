\# Diabetes Readmission Prediction - Healthcare AI for Singapore



\## Project Overview

Predicting 30-day hospital readmissions for diabetes patients using supervised ML + Gen AI, adapted for Singapore healthcare context.



\## Dataset

UCI Diabetes 130-US Hospitals (1999-2008): https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008
